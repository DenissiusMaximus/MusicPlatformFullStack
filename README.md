# Музична Платформа (Fullstack)

**Технології:**  
React, Redux Toolkit, Redux Observable, TypeScript, Tailwind CSS, React Router, Axios, ASP.NET, GraphQL

---

## Огляд
Це **повноцінний fullstack проект музичної платформи**, створений як навчальний.  
Користувачі можуть переглядати сторінки альбомів, гурту та проєкту. Авторизовані користувачі можуть **додавати, редагувати та видаляти альбоми, музику**.  

> Проект **демонстраційний** і не максимально зручний для користувачів.

---

## Функціонал
- **Фронтенд**:
  - React компоненти на TypeScript
  - Маршрутизація через React Router
  - Стан додатку через Redux Toolkit + Redux Observable epics
  - Стилізація Tailwind CSS
  - Взаємодія з бекендом через Axios
  - Використання JWT автентифікації
- **Бекенд**:
  - ASP.NET сервер з GraphQL ендпоінтами
  - Підключення бази даних для альбомів та користувачів
