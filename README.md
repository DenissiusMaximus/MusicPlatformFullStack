# Музична Платформа (Fullstack)

**Технології:**  
React, Redux Toolkit, Redux Observable, TypeScript, Tailwind CSS, React Router, Axios, ASP.NET, GraphQL

---

## Огляд
Це **fullstack проект музичної платформи**, створений як навчальний.  
Користувачі можуть переглядати сторінки альбомів, гурту та проєкту. Авторизовані користувачі можуть **додавати, редагувати та видаляти альбоми, музику**. Був створений як курсова робота з предмету **Бази даних**, тому більшість бізнес-логіки відбувається у Sql Server. У цьому проєкті наведено лише скрипт створення таблиць(без процедур та тригерів).

> Проект **демонстраційний** і не максимально зручний для користувачів.
> Деякі сторінки недоступні через інтерфейс, є зайві оновлення стейту — це залишено для простоти, часу на доопрацювання наразі немає.

---

## Функціонал
- **Фронтенд**:
  - React компоненти на TypeScript
  - Маршрутизація через React Router
  - Стан додатку через Redux Toolkit + Redux Observable epics
  - Стилізація Tailwind CSS
  - Взаємодія з бекендом через Axios
  - Використання JWT автентифікації
- **Бекенд**:
  - ASP.NET сервер з GraphQL ендпоінтами
  - Підключення бази даних для альбомів та користувачів
